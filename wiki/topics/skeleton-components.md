# Skeleton Components

## Summary [coverage: medium -- 4 sources]

`src/components/skeleton/`에 로딩 상태 스켈레톤 컴포넌트 4개가 있다. Grid, LNB, SearchForm, Tree 각 컴포넌트의 레이아웃을 미리 보여주는 플레이스홀더 UI다. `src/components/shadcn/Skeleton.tsx`의 기본 Skeleton 컴포넌트를 조합하여 구성한다.

## Component Inventory [coverage: medium -- 4 sources]

| 컴포넌트 | 용도 |
|---------|------|
| GridSkeleton | TuiGrid 로딩 전 플레이스홀더 |
| LnbSkeleton | LNB(사이드바) 로딩 전 플레이스홀더 |
| SearchFormSkeleton | SearchForm 로딩 전 플레이스홀더 |
| TreeSkeleton | Tree 컴포넌트 로딩 전 플레이스홀더 |
